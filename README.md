# Disk_usage: a user manual

This a user's manual on the common disk storage for the **teobio** group at the Department of Chemistry and Molecular Biology, University of Gothenburg.

The user's manual consist of rules that all users of the disk should adhere to, tips on useful Linux commands and guidelines on how to do backup.

## Content

* [Basic rules](rules.md)
* [Linux commands](linux.md)
* [Tips and tricks](tips.md)
* [Backup](backup.md)


## Path and acess

The disk is located on

    /priv/c3-teobio/

on the `hebbe` machine at C3SE.

Access to this disk is granted through SUPR.
